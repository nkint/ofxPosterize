What

Extract main colors from an image. Give in input a multicolored image and get as output a palette of main colors. For doing this you have to perform a kind of data reduction: from a dataset of lots of color extract the main ones. The extracted colors could not be in the initial dataset because some kind of averge is performed.

How

How does it works? It implies some different concepts. Color palette are for designer, and image segmentation is for digital signal processing guys.. very different people that uses very different tools. The best interface is that one that is build with the best tool tailored to a particular concept. So the image processing and the segmentation math is done in C++ with OpenCV library. But make a nice interface usable by designers that works with colors is a little bit problematic with plain C++ (for example make a selectable text). But it is really easy to do it with javascript. So this color extractor application works first in C++ with openframeworks for doing the dirty math, and then a little .js file is build and used for a comfortable html interface.
